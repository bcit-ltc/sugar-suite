Description

Creates interactive charts from HTML tables using Plotly.js. Supports grouped bar charts and pie charts with automatic color theming and toggle functionality to show/hide source data.

Usage

Create HTML tables with specific classes to generate charts. The tables are automatically converted to interactive charts with toggle buttons to view the original data.

Available Options

  • CSS Classes:
    • .bar-grouped - Creates grouped bar chart
    • .pie - Creates pie chart
    • .flip - Flips data orientation
    • .static - Creates static (non-interactive) chart
  • Features:
    • Interactive charts with hover effects
    • Automatic color theming
    • Toggle buttons to show/hide source data
    • Responsive design
    • Legend and axis labels

Notes

  • Charts are automatically generated from HTML tables
  • Use .bar-grouped for grouped bar charts
  • Use .pie for pie charts
  • Use .flip to transpose data orientation
  • Use .static for non-interactive charts
  • Toggle buttons allow viewing original table data
  • Charts use automatic color theming from CSS variables
  • Plotly.js library is loaded automatically when needed
  • Charts are responsive and work on all screen sizes
  • Interactive features include hover effects and legend toggling
  • Source tables are hidden by default but can be toggled
