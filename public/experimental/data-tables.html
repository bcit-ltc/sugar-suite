Data Tables

Description

Enhances tables with advanced DataTables functionality including sorting, searching, pagination, and filtering. Provides a powerful interface for managing large datasets with professional table controls.

Usage

Add the .data-table class to any table to enable DataTables functionality. Additional classes control specific features like pagination and sorting.

Available Options

  • CSS Classes:
    • .data-table - Main container (required)
    • .paging - Enables pagination
    • .static - Disables sorting
    • .info - Shows table information
  • Features:
    • Column sorting (click headers)
    • Search functionality
    • Pagination controls
    • Length selection (10, 25, 50, All)
    • Responsive design

Notes

  • DataTables automatically loads when .data-table class is present
  • Default features include sorting and search functionality
  • Use .paging to enable pagination controls
  • Use .static to disable sorting (useful for ordered data)
  • Use .info to show table information (e.g., "Showing 1 to 10 of 25 entries")
  • Length menu allows users to choose how many rows to display
  • Search functionality works across all columns
  • Tables are responsive and work well on mobile devices
  • DataTables library is loaded automatically when needed
  • All features are accessible via keyboard navigation
