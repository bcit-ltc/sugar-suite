Fancy Images

Description

Provides advanced image effects including widest responsive behavior, saturation filters, and shape-outside text wrapping. These features enhance visual presentation and create more engaging layouts.

Usage

Add specific CSS classes to images to enable different fancy effects. Each class provides a different visual enhancement for images.

Available Options

  • CSS Classes:
    • .widest - Responsive width adjustment with ghost spacing
    • .saturate - Increases color saturation (200%)
    • .shape-outside - Enables text wrapping around image shape
  • Features:
    • Automatic responsive behavior
    • CSS filter effects
    • Text flow around image shapes
    • Dynamic spacing adjustments

Notes

  • .widest creates responsive behavior with automatic ghost spacing
  • .saturate increases color saturation to 200% for more vibrant images
  • .shape-outside enables text to wrap around the actual shape of the image
  • Multiple classes can be combined for enhanced effects
  • Widest images automatically adjust on window resize
  • Saturation effects work with any image format
  • Shape-outside requires the image to have a transparent or distinct background
  • All effects are CSS-based and performant
  • Effects work well with other image features like click-to-enlarge
  • Fancy images maintain accessibility and responsive behavior
