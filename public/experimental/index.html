Sugar Suite Experimental Features

Experimental features that are still in development. These features may change or be removed in future versions.

Experimental Features

Experimental Interactive

  • Tooltip - Hover tooltips
  • New Window - Small popup windows for supplementary content

Experimental Utilities

  • Timer - Countdown timers with visual feedback
  • Fullscreen - Fullscreen mode toggle
  • Night Mode - Dark mode overlay for comfortable reading
  • Experimental Loader - Automatic loading of experimental CSS and JavaScript files

Experimental Navigation

  • On-Page Navigation - Automatic table of contents generation
  • Page Navigation - D2L page navigation controls

Experimental Data & Media

  • Data Tables - Sortable/searchable tables with advanced features
  • Charts - Data visualization using Plotly
  • Fancy Images - Advanced image effects (zoom, filters, etc.)

Experimental Developer Tools

  • Debug Mode - Visual debugging indicators and development information
  • APA Formatting - Academic citation styling and document formatting
  • Lined Paper - Notebook-style background effect

About Experimental Features

These features are experimental and may:

  • Change significantly in future versions
  • Be removed entirely
  • Have limited browser support
  • Require additional dependencies

Use with caution in production environments.
