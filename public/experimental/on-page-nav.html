On-Page Navigation

Description

Automatically generates a table of contents (TOC) for pages with sufficient headings. The TOC appears after the first h1 heading and provides quick navigation to different sections of the page.

Usage

On-page navigation is automatically activated when a page contains 5 or more headings (h1-h6). The TOC is generated automatically and appears after the first h1 heading.

Automatic Features

  • Threshold: Requires 5 or more headings to activate
  • Placement: Appears after the first h1 heading
  • Content: Includes all headings (h1-h6) in hierarchical order
  • Navigation: Clickable links that scroll to sections
