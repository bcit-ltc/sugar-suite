Page Navigation

Back to Home

Description

Provides D2L (Desire2Learn) page navigation controls including Previous/Next page functionality and a "Back to top" button. These controls enhance navigation within D2L learning management systems.

Usage

Page navigation controls are automatically applied in D2L environments. They provide Previous/Next page functionality and a "Back to top" button for enhanced navigation.

Available Features

  • Previous Page: Navigate to the previous page in the sequence
  • Next Page: Navigate to the next page in the sequence
  • Back to Top: Quick return to the top of the current page
  • D2L Integration: Works within D2L learning management systems
