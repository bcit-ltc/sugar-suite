Description

Provides countdown timers with visual feedback for timed activities and assessments. Timers automatically change colors based on remaining time and perform dramatic exit animations when time expires.

Usage

Add data-timer="minutes" attribute to any container element. The timer will automatically start when clicked and provide visual feedback throughout the countdown.

Available Options

  • Data Attributes:
    • data-timer="minutes" - Sets timer duration in minutes
  • Visual Feedback:
    • Green: More than 60% time remaining
    • Orange: 30-60% time remaining
    • Red: Less than 30% time remaining
    • Black: Time expired

Notes

  • Timers require data-timer attribute with duration in minutes
  • Timer starts when clicked by the user
  • Visual feedback changes color based on remaining time percentage
  • Dramatic exit animation plays when time expires
  • Timers work within any container including activities and assessments
  • Multiple timers can be used on the same page
  • Timer duration should be appropriate for the task complexity
  • Consider providing clear instructions before starting the timer
  • Use timers to create urgency and focus for timed activities
  • Timers enhance the learning experience by providing structure
  • Visual feedback helps users manage their time effectively
