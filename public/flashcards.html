Flashcards

Description

Transforms tables into interactive flashcard sets with flipping animations, navigation controls, shuffle functionality, and responsive design. Perfect for studying and memorization activities.

Usage

Create a table with class .flashcards where each row represents a card with two cells (front and back). The table is automatically converted into an interactive flashcard interface.

Available Options

  • CSS Classes:
    • .flashcards - Main container (required)
    • .no-number - Hides the card counter
    • .no-nav - Hides navigation controls (enables swipe gestures)
    • .top-left, .top-center, .top-right - Card content alignment
    • .middle-left, .middle-center, .middle-right - Card content alignment
    • .bottom-left, .bottom-center, .bottom-right - Card content alignment
  • Data Attributes:
    • data-height="number" - Set custom card height in pixels (default: 500px)
    • data-justify-content="value" - CSS justify-content value for card content
    • data-align-items="value" - CSS align-items value for card content

Notes

  • Each table row becomes a flashcard with two sides
  • First cell is the front, second cell is the back
  • Navigation controls: Previous, Flip, Next, Flip All, Shuffle, Reset, View Table
  • Keyboard navigation: Arrow keys for navigation and flipping
  • Swipe gestures work when .no-nav is used
  • Cards can contain any HTML content (text, images, lists, etc.)
  • Custom height can be set using data-height attribute
  • Content alignment classes control how content is positioned within cards
  • Original table is preserved and can be toggled with "View as Table" button
  • Cards maintain their state during navigation and shuffling
