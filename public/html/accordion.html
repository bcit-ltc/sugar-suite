Accordion

Description

Creates collapsible content sections where clicking on headings (h2, h3, etc.) toggles the visibility of content below. Also includes reveal functionality for click-to-show content.

Usage

Wrap content in a container with class .accordion. The first child element type becomes the toggle trigger (typically h2). Content following each trigger is automatically wrapped and hidden.

Available Options

  • CSS Classes:
    • .accordion - Main container
    • .accordion-alt - Alternative styling without arrow indicators
    • .no-toggle - Removes the toggle-all button
    • .collapsing - Makes accordion behave like tabs (only one section open at a time)
  • Data Attributes:
    • data-button="text" - Custom text for reveal button (default: "Reveal")
    • data-min-text="number" - Minimum characters required before reveal button activates
    • data-placeholder="text" - Placeholder text for textarea (when using min-text)
    • data-rows="number" - Number of rows for textarea (default: 5)

Notes

  • The first child element type (h2, h3, etc.) becomes the toggle trigger
  • Content following each trigger is automatically wrapped in ._bellow containers
  • Accordion supports keyboard navigation (Enter/Space keys)
  • The toggle button (top-left) opens all closed sections or closes all open sections
  • Reveal functionality works independently of accordion structure
  • Use .accordion-alt for a cleaner look without arrow indicators
