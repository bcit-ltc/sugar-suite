Containers

Description

Provides styled containers for different types of content including activities, assessments, notes, warnings, and examples. Each container has distinct styling to help organize and highlight content appropriately.

Usage

Wrap content in the appropriate container class to apply specific styling and visual treatment. Each container type has its own purpose and styling.

Available Containers

  • .activity - For learning activities and exercises
  • .assessment - For assessments and evaluations
  • .note - For general notes and information
  • .warning - For important warnings and cautions
  • .example - For examples and demonstrations
  • Additional Activity Types:
    • .case - For case studies
    • .discussion - For discussion prompts
    • .group-activity - For group work activities
    • .outcome - For learning outcomes
    • .reading - For reading assignments
    • .reflection - For reflection exercises
    • .review - For review activities
    • .doing - For hands-on activities
    • .video-activity - For video-based activities with play icon

Notes

  • Each container type has distinct visual styling and purpose
  • Additional activity types (.case, .discussion, .group-activity, etc.) provide specialized styling
  • All activity types share the same base styling with different icons
  • Containers can be nested for complex content organization
  • Use appropriate containers to help users understand content importance
  • Containers work well with other layout components
  • All containers are responsive and work on all screen sizes
  • Containers provide consistent spacing and visual hierarchy
  • Use semantic container types for better content organization
  • Containers can contain any HTML content including other components
