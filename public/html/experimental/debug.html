Debug Mode

Description

Provides visual debugging indicators and development information for content creators and developers. Debug mode shows development notes, content status, and other helpful information for content management.

Usage

Add .debug class to any container element to enable debug mode. Debug mode is primarily for content development and management purposes.

Available Options

  • CSS Classes:
    • .debug - Enables debug mode for the container
  • Debug Features:
    • Visual debugging indicators
    • Development information display
    • Content status indicators
    • Development notes visibility
