Fullscreen

Description

Provides fullscreen mode toggle functionality for enhanced focus and immersive learning experiences. Users can toggle fullscreen mode using buttons or keyboard shortcuts.

Usage

Fullscreen functionality is automatically available when experimental features are loaded. A fullscreen toggle button is automatically added to the page.

Available Controls

  • Fullscreen Button: Automatically added to the page
  • Keyboard Shortcuts:
    • F11 - Toggle fullscreen mode
    • Alt+Backspace - Toggle fullscreen mode
    • Ctrl+R - Toggle fullscreen mode
