Lined Paper

Description

Provides a lined paper background effect that creates a notebook or legal pad appearance. Perfect for note-taking activities, writing exercises, or content that should feel more personal and handwritten.

Usage

Add .lined-paper class to any container element to apply the lined paper background effect. This creates a notebook-style appearance with horizontal lines.

Available Options

  • CSS Classes:
    • .lined-paper - Applies lined paper background effect
  • Visual Effect:
    • Notebook-style horizontal lines
    • Legal pad appearance
    • Personal, handwritten feel
    • Enhanced writing experience
