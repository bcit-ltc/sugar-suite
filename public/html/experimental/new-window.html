New Window

Description

Opens links in small, popup-style windows with minimal browser chrome. Perfect for displaying additional content, references, or supplementary information without navigating away from the main content.

Usage

Add the new-window class to any anchor tag. When clicked, the link will open in a small popup window instead of a new tab.

Available Options

  • CSS Classes:
    • .new-window - Opens link in small popup window
  • Window Features:
    • Small size (200x200 pixels)
    • Minimal browser chrome
    • No menu bar, toolbar, or status bar
    • No scrollbars
    • Positioned off-screen initially

Notes

  • New window links open in small popup windows (200x200 pixels)
  • Windows have minimal browser chrome for focused content
  • Perfect for displaying additional information without losing context
  • Use sparingly as popup blockers may interfere
  • Consider accessibility - some users may have popup blockers enabled
  • Provide clear indication that links will open in new windows
  • Test across different browsers and devices
  • Consider mobile users who may not handle popups well
  • Use for supplementary content, not essential information
  • Provide alternative access methods for important content
