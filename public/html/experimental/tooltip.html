Tooltip

Description

Creates interactive tooltips using Tippy.js library. Tooltips can contain rich HTML content and are triggered by click events. Perfect for providing additional information, definitions, or context without cluttering the main content.

Usage

Add the .tooltip class to elements that should trigger tooltips. Tooltip content can be provided in several ways: adjacent elements, data attributes, or content IDs.

Available Options

  • CSS Classes:
    • .tooltip - Main trigger element
    • .tooltip-content - Content container
  • Data Attributes:
    • data-tooltip-content - Direct content
    • data-content-id - ID of content element
    • data-tooltip-props - JSON configuration

Notes

  • Tooltips are triggered by click events by default
  • Content can be provided via adjacent elements, data attributes, or content IDs
  • Tooltips support rich HTML content including images, lists, and formatting
  • Custom properties can be set via data-tooltip-props attribute
  • Tooltips are interactive and can contain clickable elements
  • Placement is automatic but can be customized
  • Tooltips work well in all content contexts including tables and lists
  • Content is hidden by default and shown when tooltip is triggered
  • Tooltips are accessible and keyboard navigable
