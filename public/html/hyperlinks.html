Hyperlinks

Description

Enhances hyperlinks with automatic target="_blank" behavior, external link detection, file type handling, and proper accessibility attributes. Automatically opens links in new tabs for better user experience.

Usage

Create standard HTML anchor tags. The system automatically enhances them with appropriate target attributes, external link detection, and file type handling.

Automatic Features

  • New Tab Opening: All links open in new tabs by default
  • External Link Detection: Automatically detects external links
  • File Type Handling: Special handling for common file types
  • Download Links: Automatic title attributes for download links
  • Email Links: Fixes for malformed email addresses
