Knowledge Check

Description

Creates interactive quizzes with multiple question types including multiple choice, multiple select, fill-in-the-blank, ordering, and matching. Provides immediate feedback and scoring.

Usage

Wrap questions in a container with class .knowledge-check or .self-test. Use specific HTML structure and special syntax for different question types.

Available Options

  • CSS Classes:
    • .knowledge-check or .self-test - Main container
    • .no-score - Hides the score display
    • .development - Removes required validation for testing
    • .no-list-style - Removes list styling from options
    • .multicolumn-2 - Displays options in 2 columns
    • .multicolumn-3 - Displays options in 3 columns
  • Special Syntax:
    • * - Marks correct answers (prefix)
    • @ - Indicates feedback text (prefix)
    • ul - Creates unordered list (random order)
    • ol - Creates ordered list (fixed order)

Notes

  • Use * to mark correct answers
  • Use @ to provide feedback text
  • ul creates random order, ol creates fixed order
  • Fill-in-the-blank uses [brackets] around the correct answer in the text
  • Ordering questions use a simple list without * prefixes
  • Matching questions use = signs to connect pairs (e.g., "JavaScript = Web Development")
  • Multi-column layouts work with 2 or 3 columns (requires multiple questions to see effect)
  • Development mode removes validation requirements for testing
  • No-score mode hides the score display
  • Questions are automatically numbered unless using custom numbering
