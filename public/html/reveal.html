Reveal

Description

Creates click-to-show content functionality with optional text input requirements. Perfect for answers, solutions, hints, or any content that should be revealed on demand. Supports both simple reveal and active reveal with user input requirements.

Usage

Wrap content in a container with class .reveal or .active-reveal. The system automatically creates reveal buttons and handles the show/hide functionality.

Available Options

  • CSS Classes:
    • .reveal - Simple click-to-show content
    • .active-reveal - Requires user input before revealing
  • Data Attributes:
    • data-button="text" - Custom text for reveal button (default: "Reveal")
    • data-min-text="number" - Minimum characters required for active-reveal
    • data-placeholder="text" - Placeholder text for textarea
    • data-rows="number" - Number of rows for textarea (default: 5)

Notes

  • Reveal content is hidden by default and shown with slide animation
  • Use .reveal for simple click-to-show functionality
  • Use .active-reveal to require user input before revealing
  • Custom button text with data-button attribute
  • Minimum text length with data-min-text attribute
  • Custom placeholder text with data-placeholder attribute
  • Custom textarea rows with data-rows attribute
  • Reveal content works well with other interactive elements
  • Content is automatically shown when printing
  • Perfect for progressive disclosure of information
