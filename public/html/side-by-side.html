Side-by-Side Layout

Description

Creates responsive two-column layouts that automatically stack on mobile devices. Perfect for comparing content, showing before/after examples, or organizing related information.

Usage

Wrap content in a container with class .side-by-side. Each direct child becomes a column. Content automatically stacks vertically on mobile devices.

Available Options

  • CSS Classes:
    • .side-by-side - Main container for two-column layout
    • .align-top - Aligns columns to the top
    • .align-bottom - Aligns columns to the bottom
  • Responsive Behavior:
    • Desktop: Two equal-width columns
    • Mobile: Stacked vertically
    • Automatic spacing and alignment

Notes

  • Each direct child of .side-by-side becomes a column
  • Columns have equal width on desktop screens
  • Content automatically stacks vertically on mobile devices
  • Use .align-top to align columns to the top
  • Use .align-bottom to align columns to the bottom
  • Default alignment centers columns vertically
  • Side-by-side layouts can be nested within other layouts
  • Works well with figures, activities, and other content types
  • Responsive design ensures good mobile experience
  • Perfect for comparing content, showing processes, or organizing information
  • Maintains consistent spacing and visual hierarchy
