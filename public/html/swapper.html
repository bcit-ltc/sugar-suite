Swapper

Description

Creates sequential content navigation with Previous/Next buttons and automatic height adjustment. Perfect for step-by-step processes, tutorials, or any content that needs to be viewed sequentially. Automatically handles YouTube video pausing and smooth transitions.

Usage

Wrap sequential content sections in a container with class .swapper. Each direct child becomes a "slide" that can be navigated through using the automatically generated Previous/Next buttons.

Available Options

  • CSS Classes:
    • .swapper - Main container for sequential content
  • Automatic Features:
    • Previous/Next navigation buttons
    • Slide counter display (e.g., "2/5")
    • Automatic height adjustment
    • Smooth fade transitions
    • YouTube video pausing
    • Scroll-to-top on navigation
    • Circular navigation (Next from last goes to first)

Notes

  • Each direct child of .swapper becomes a navigable slide
  • Navigation buttons are automatically generated
  • Slide counter shows current position (e.g., "2/5")
  • Height automatically adjusts to accommodate the tallest slide
  • Smooth fade transitions between slides
  • YouTube videos are automatically paused when navigating away
  • Page scrolls to top of swapper when navigating
  • Navigation is circular (Next from last slide goes to first)
  • Works with any HTML content including interactive elements
  • Perfect for tutorials, step-by-step processes, and sequential content
  • Maintains consistent spacing and visual hierarchy
  • Responsive design works on all screen sizes
