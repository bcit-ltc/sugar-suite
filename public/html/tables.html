Tables

Description

Enhances tables with automatic striping, responsive scrolling, column alignment, and minimal styling options. Automatically applies appropriate styling based on table size and content.

Usage

Create standard HTML tables. The system automatically enhances them with responsive behavior, striping for large tables, column alignment based on header cell classes, and overflow scrolling.

Available Options

  • CSS Classes:
    • .striped - Applied automatically to large tables (5+ rows and columns)
    • .unstriped - Prevents automatic striping
    • .unstyled - Removes all automatic styling
    • .overflow-x - Applied automatically for horizontal scrolling
    • .minimal - Applies minimal table styling
    • .borderless - Removes all borders
    • .compact - Reduces font size and padding for dense tables
    • .left, .center, .right - Column alignment classes
    • .top, .middle, .bottom - Vertical alignment classes
  • Automatic Features:
    • Responsive horizontal scrolling wrapper
    • Automatic striping for large tables
    • Column alignment based on header classes
    • Minimal styling enhancements

Notes

  • Tables with 5+ rows and columns automatically get striped styling
  • All tables are wrapped in responsive scrolling containers
  • Column alignment is applied based on header cell classes
  • Use .left, .center, or .right on header cells to align entire columns
  • Use .top, .middle, or .bottom for vertical alignment
  • .unstriped prevents automatic striping
  • .unstyled removes all automatic enhancements
  • .minimal applies clean, minimal styling
  • .borderless removes all borders for a clean look
  • .compact reduces font size and padding for dense tables
  • Complex tables with rowspan/colspan are excluded from column alignment
  • Flashcard tables are excluded from responsive wrapping
  • Tables automatically adapt to screen size with horizontal scrolling
