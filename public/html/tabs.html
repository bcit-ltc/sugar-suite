Tabs

Description

Creates a tabbed interface where content is organized into clickable tabs. Each tab contains related content that can be accessed by clicking the tab headers.

Usage

Wrap content in a container with class .tabs. Use headings (h2, h3) as tab titles. Content following each heading becomes tab content.

Available Options

  • CSS Classes:
    • .tabs - Main container for tabbed interface
    • .nav-left - Places navigation on the left side (default)
    • .nav-top - Places navigation at the top
    • .nav-bottom - Places navigation at the bottom
    • .mono-color - Uses single color scheme instead of pastel colors
    • .no-title - Hides the heading text in tab content (heading still used for tab button)
  • Data Attributes:
    • data-color="color" - Custom background color for individual tabs
    • data-font-color="color" - Custom text color for individual tabs

Notes

  • Tab navigation is automatically generated from headings
  • Use "Tabs text: [custom text]" after headings to override tab labels
  • Tab colors can be customized per tab using data attributes
  • Tabs work well with other interactive elements
  • Use .nav-top or .nav-bottom to control tab placement
  • Add class="no-title" to headings to hide them in content while keeping tab buttons
