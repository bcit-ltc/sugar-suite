Lists

Description

Provides comprehensive styling for ordered lists, unordered lists, and definition lists with consistent spacing, indentation, and visual hierarchy.

Usage

Use standard HTML list elements. The system provides consistent styling for all list types with proper spacing and indentation.

Available List Types

  • Unordered Lists: ul with li elements
  • Ordered Lists: ol with li elements
  • Definition Lists: dl with dt and dd elements
  • Nested Lists: Lists within lists

Available CSS Classes

  • List Style Classes:
    • .unstyled - Removes list markers
    • .disc - Disc markers for unordered lists
    • .circle - Circle markers for unordered lists
    • .square - Square markers for unordered lists
    • .decimal - Decimal numbers for ordered lists
    • .lower-alpha - Lowercase letters (a, b, c)
    • .upper-alpha - Uppercase letters (A, B, C)
    • .lower-roman - Lowercase Roman numerals (i, ii, iii)
    • .upper-roman - Uppercase Roman numerals (I, II, III)
    • .legal - Legal-style numbering (1.1, 1.2, 2.1)

Notes

  • Lists automatically have proper spacing and indentation
  • Nested lists maintain proper hierarchy and indentation
  • Definition lists are perfect for glossaries and term definitions
  • Use list style classes to customize appearance (disc, circle, square, decimal, etc.)
  • .legal class creates hierarchical numbering (1.1, 1.2, 2.1)
  • .unstyled removes markers for clean navigation lists
  • Lists can contain any HTML content including links, images, and other elements
  • Use semantic HTML elements for better accessibility
  • Lists scale appropriately across different screen sizes
  • Consistent styling ensures readability and visual hierarchy
