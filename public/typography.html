Typography

Description

Provides comprehensive typography styling including headings, text formatting, emphasis, blockquotes, and other text elements with consistent spacing and hierarchy.

Usage

Use standard HTML elements for typography. The system provides consistent styling for headings, paragraphs, emphasis, and other text elements.

Available Elements

  • Headings: h1, h2, h3, h4, h5, h6 or .heading1, .heading2, .heading3, .heading4, .heading5, .heading6
  • Text Formatting: p, strong, em, mark, small, del, ins
  • Blockquotes: blockquote with optional citation
  • Code: code, pre, kbd
  • Other: hr, address, abbr, dfn

Notes

  • Headings create a clear hierarchy and improve readability
  • Use standard h1-h6 elements or alternative .heading1-.heading6 classes
  • Alternative heading classes are useful when you need heading styling on non-heading elements
  • Use <strong> for important text, <em> for emphasis
  • Blockquotes are perfect for highlighting important quotes or information
  • Code elements maintain proper formatting and are easy to read
  • All typography elements have consistent spacing and styling
  • Use semantic HTML elements for better accessibility
  • Typography scales appropriately across different screen sizes
